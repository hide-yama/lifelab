### Day 3（2026-03-11）

| 項目 | 記録 |

| 回復感 | 3/5 |
| 体の状態 | 肩首こり、背中・腰の凝り、胃腸の重さ（脂っこい食事+白ワイン） |
| 夜コーヒー | 22時台 |
| 気分 | 普通 |
| 介入実施 | クレアチン ✅（Day 3） |

**会話で出てきたこと**:
- 睡眠が6.5時間と短めで、体に眠さが残っている
- 体全体が少し凝りがち（肩首に加えて背中・腰も）
- 昨夜は脂っこい食事＋白ワインで胃腸が重い

**追加ヒアリング**:
- 白ワイン グラス2〜3杯
- 就寝前ショート動画: 少しだけ
- 朝ショート動画: 少し見た
- ラジオ体操: やってない
- 水分: まあまあ（ペットボトル半分くらい）
- 朝食: 食べた（タンパク質含む）
- 集中力: まあまあ
- エネルギーレベル: 低め
- モチベーション: 普通
- 睡眠の質: 浅かった（何度も目が覚めた、熟睡感なし）

**トレーナーの所感**:
- 回復感3/5はDay 2と横ばいだが、中身が違う。睡眠が1時間短く、質も浅い。白ワイン2〜3杯＋脂っこい食事が睡眠の質を下げた可能性が高い
- Day 1もDay 3も飲酒した日に胃腸の重さ＋睡眠の質低下が出ている。飲酒→睡眠悪化のパターンが見え始めている
- エネルギーレベル「低め」は睡眠の質・量のダブルパンチの結果と見るのが妥当
- 朝のショート動画は「少し」で済んでいる。Day 1-2の記録と比較して改善傾向かどうか要追跡
- ラジオ体操をスキップしている。エネルギーが低い日ほどやったほうがいいが、強制はしない
- 朝食を摂れているのは良い。クレアチンも継続できている

**夕方の問診（19時頃）**:

| 項目 | 記録 |
|---|---|
| 頭のクリアさ | 4/5 |
| 集中力の持続 | 前半低め→後半良い（体感3→4） |
| 午後の眠気 | 14-15時頃まで軽い眠気あり、後半は解消 |
| ショート動画 | 結構見た（休憩のつもりで中断→戻れなくなるパターン） |
| エネルギー推移 | 朝「低め」→後半で上がった |
| ラジオ体操後の凝り | 少し楽になった |

**会話で出てきたこと**:
- 仕事の後半でのめり込めた。腰の重い仕事だったが、エンジンがかかってからは調子に乗って集中できた
- ショート動画は休憩のつもりで見始めて結構見てしまった（Day 2と同パターン）
- 体調が悪い日とショート動画の量には関係があると本人も感じている

**トレーナーの所感**:
- 頭のクリアさ4/5はDay 2と同じ。睡眠が浅く短かった割に脳のパフォーマンス自体は維持できている。クレアチンの効果はまだ判断できないが、悪くない
- 「前半ダメ→後半のめり込む」パターンが出た。腰が重い仕事ほど、一度エンジンがかかると没頭できるタイプ。問題はエンジンがかかるまでの時間にショート動画で逃げてしまうこと
- **飲酒→睡眠浅い→エネルギー低い→ショート動画で逃避しやすい**の連鎖が3日分のデータで浮かび上がってきた。Day 2（飲酒少なめ）は動画中断がありつつも眠気なし。Day 1・Day 3（飲酒多め）は眠気あり＋動画多め
- ラジオ体操で凝りが少し楽になっている。やらなかった朝との比較になっており、体操の効果の小さなエビデンス
- 次の介入変数の候補として「飲酒量と睡眠の質の関係」を意識的に追跡する価値がある。ただし現在はクレアチンの実験期間中なので、介入はせず観察のみ

### Day 4（2026-03-12）

| 項目 | 記録 |
|---|---|
| 就寝→起床 | 3時台 → 9時台（約5時間50分） |
| 回復感 | 記録なし（朝の問診スキップ） |
| 介入実施 | クレアチン ✅（Day 4） |

**夕方の問診（18時頃）**:

| 項目 | 記録 |
|---|---|
| 頭のクリアさ | 4/5 |
| 集中力の持続 | 3/5 |
| 午後の眠気 | 軽い（14時台） |
| ショート動画 | がっつり（1時間以上） |
| ラジオ体操 | やってない |

**会話で出てきたこと**:
- 13〜17-18時はいい感じに仕事をさばけた。スロースタートだったが後半は調子が出た
- ただしタスクが整理されないまま「目の前のことにもがく」状態で処理したため、精神的に疲れた
- **タスクの並べ方の重要性を実感**: 散らばった状態 vs 一列に並べて1つずつ、では同じ量でもストレスとパフォーマンスが全然違う
- 睡眠5時間50分。昨晩の研究（lifelab）が楽しくて夜更かしした結果
- 睡眠不足 → エネルギー低下 → 夜の支度もだらける → 悪循環を自覚
- 「悪いサイクルは悪いサイクルを呼び、いいサイクルはいいサイクルを呼ぶ。いいサイクルにどうレールを乗せるかが大事」

**トレーナーの所感**:
- 頭のクリアさ4/5は4日連続で安定。クレアチンの効果はまだ判断できないが、睡眠5時間50分でも4を維持しているのは注目に値する
- 集中力3/5、眠気「軽い」は睡眠不足を考えると妥当。Day 3と同様「前半ダメ→後半調子が出る」パターン
- **ショート動画がっつり（1時間以上）は4日間で最多**。睡眠不足→エネルギー低下→逃避のパターンが再現。Day 2（睡眠まあまあ）< Day 1・3（飲酒で質低下）< Day 4（絶対量不足）と、睡眠の質・量が悪いほどショート動画が増える傾向がより明確になった
- 本人の「タスクの並べ方」の気づきは重要。これはこころ研究室やデジタルデトックスとも繋がる。タスクが散乱 → 脳の負荷増 → 逃避しやすくなる、という構造
- **睡眠を削った原因が「研究が楽しかった」というポジティブな理由であっても、翌日のパフォーマンスへの影響は同じ**。1日1研究テーマルール（終わったら今日は終わり）はこのためにある
