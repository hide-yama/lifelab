**ステータス**: 実験中 — クレアチン介入の経過観測（2026-03-09〜）

食事・運動・睡眠・姿勢を総合的に扱う。
「脳の物理的エネルギー不足」を解消することが最上位の目的。

## 現状の課題

- 運動ほぼなし（慢性的な肩こり・首こり）

## 改善ロードマップ（優先順）

6. ストレッチの習慣化

## 毎日の問診（PDCAシステム）

毎朝のセッション冒頭でAIトレーナーが問診を行い、体調の変化を追跡する。
データとアドバイスをセットで記録し、経過観察として振り返れる形にする。

### 朝の問診

#### 事実の記録

1. **就寝→起床**（時刻）
2. **睡眠**
    - 中途覚醒（なし / あり（回数や状況を簡単に））
    - 起床時の眠気（残っていない / 少し残っている / 強く残っている）
3. **前夜の摂取**
    - コーヒー（時間帯 or 飲まなかった）
    - 飲酒（種類・量 or 飲まなかった）
4. **介入変数の実施**（やった / やらなかった）
5. **習慣の追跡**（※現フェーズに応じて項目を更新する）
    - 朝散歩（した / しなかった）
    - 朝のショート動画（見てない / 10分以内 / 30分程度 / 1時間以上）
    - 朝の覚醒手段（小説 / ショート動画 / 自然起床 / その他）

#### 主観の3軸（体・頭・心）

6. **体**（回復感）：睡眠で体が戻ったか
    - 5=スッキリ（体が軽い、すぐ動ける）
    - 4=回復した実感がある（昨日の疲れは取れた、普通に動ける）
    - 3=どちらとも言えない（悪くはないが、回復したとも言い切れない）
    - 2=足りてない（だるさが残る、体が重い）
    - 1=ぐったり（起き上がるのがつらい、動きたくない）
7. **頭**（クリアさ）：頭が起きているか
    - 5=シャープ（思考が冴えている、判断が速い）
    - 4=クリア（普通に考えられる）
    - 3=普通（特に良くも悪くもない）
    - 2=重い（霧がかかっている、考えがまとまりにくい）
    - 1=ぼんやり（何も考えたくない、頭が働かない）
8. **心**（気分）：気持ちの状態
    - 5=前向き（やる気がある、明るい気持ち）
    - 4=やや前向き（悪くない、少し前向き）
    - 3=フラット（特に良くも悪くもない）
    - 2=やや重い（少し気が乗らない、腰が重い）
    - 1=沈んでいる（気分が落ちている、やる気が出ない）

#### 補足

9. **体の状態**（自由記述）: 肩首こり、目の疲れ、背中、胃腸 など
10. **会話**: 選択式の後に「昨日と比べて変わったことや気になることは？」と聞く。本人の言語化から気づきを拾い、アドバイスに繋げる

### 夕方の問診

#### 主観の3軸（朝との変化を見る）

1. **体**（疲労度）：今の体の疲れ具合
    - 5=まだ元気（体力が残っている）
    - 4=普通に疲れた（通常の1日分の消耗）
    - 3=やや疲れた（少し消耗が大きい）
    - 2=かなり疲れた（早く休みたい）
    - 1=限界（体が動かない）
2. **頭**（クリアさ）：朝と同じ基準
3. **心**（気分）：朝と同じ基準

#### 事実の記録

4. **午後の眠気**（なし / 軽い / 強い）
5. **ショート動画**（日中）（見てない / 少し（10-30分）/ かなり（30分以上））

#### 補足

6. **会話**: 「今日1日やってみてどうでしたか？」と聞く。仕事の状況や集中力の話は会話の中で自然に拾う

### 問診の進め方

- **形式**: AskUserQuestion（ターミナル上の選択式UI）を使う。テキストで選択肢を並べるのではなく、必ず選択UIで出す
- **最初に聞く**: 「がっつり問診しますか？ さくっと済ませますか？」
  - **がっつり**: 基本項目＋深掘り（飲酒量、ショート動画、朝の過ごし方、集中力、エネルギー、モチベーション、睡眠の質など）＋1日のアドバイス
  - **さくっと**: 基本項目（就寝起床・回復感・体の状態・夜コーヒー・気分・介入実施）のみ
- **会話パート**: 選択式の後に自由記述で補足してもらう

### ルール

- **介入**: 1変数ずつ、最低2週間ずつ
- **振り返り**: 週末にデータを見て傾向を確認
- **記録**: 月別ファイルに蓄積（`問診ログ/YYYY-MM.md`）
