# 収支振り返りシステム

独自のお金のカテゴリー理論に基づいて、週次で収支を振り返る仕組みを作る研究。

## 現状
- 家計の記録は銀行・カードの明細のみ。家計簿アプリやスプレッドシートでの管理はしていない
- 独自のお金のカテゴリー理論がAppleメモにある（未移植）
- 支出が分類された状態で見られる仕組みはまだない

## やりたいこと
- 独自のカテゴリー理論に照らし合わせて、使ったお金がどの分類に当たるかを振り返る
- カテゴリーごとの割合が正常かどうかを確認する
- 全体の支出量が適切かどうかを考えるきっかけにする
- あゆさんと週次で話し合いながら振り返る

## 次のステップ
1. Appleメモにあるカテゴリー理論をこの研究フォルダに移植・整理する
2. カテゴリーに沿った振り返りフォーマットを設計する
3. 来週末あゆさんと最初の振り返りを実施する
