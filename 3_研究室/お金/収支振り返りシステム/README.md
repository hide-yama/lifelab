## 4分類カテゴリー理論

**核心ルール: ノイズだけ減らす。楽しみは削らない。**

節約する／しないではなく、「意味」で分けて考える。

| 分類 | 金額目安 | 割合 | 意味 |
|---|---|---|---|
| 生存（ベース） | 約10万円 | 50% | 生きるために必要。削りすぎ注意 |
| 維持（QOL） | 約4万円 | 20% | 生活の質を守る |
| 拡張（楽しみ） | 約4.5万円 | 22.5% | 思い出に残る。ここは守る領域 |
| ノイズ（無意識） | 1.5万円以内 | 7.5% | 思い出に残らない。ここだけ意識 |

### 具体例

| ジャンル | 生存 | 維持 | 拡張 | ノイズ |
|---|---|---|---|---|
| 食べ物 | 普段の食材 | ちょっと良い食材 | 記念日の外食 | 疲れて適当にウーバー |
| コーヒー | — | 家で飲む豆 | カフェ時間 | なんとなくコンビニ |
| ワイン | — | — | 週末に楽しむ | なんとなく毎日 |
| 洋服 | — | 日常着 | こだわりデザイン・おしゃれ投資 | セール衝動買い |
| 家具・家電 | 壊れて買い替え | 生活を整える | — | — |

### 分類の境界線ガイド

4分類は明確に線引きできるものではない。同じ行為でも文脈によって分類が変わる。**大事なのは「正解」を決めることではなく、2人で「これはどれだろう？」と話すこと自体**。

#### 生存 vs 維持

| 生存 | 維持 |
|---|---|
| なければ生活が成り立たない | なくても死なないが、ないと生活の質が落ちる |
| 米、卵、トイレットペーパー | ちょっと良い調味料、柔軟剤、観葉植物 |
| **判断基準**: 「これがなかったら困る？」→ Yes=生存 / 「なくても困らないけど、あると暮らしが整う」→ 維持 |

#### 維持 vs 拡張

| 維持 | 拡張 |
|---|---|
| 日常を「整える」もの | 日常を「超える」もの |
| 繰り返し使う・消耗する | 体験・記憶として残る |
| 家で飲むコーヒー豆、日常着 | カフェでゆっくりする時間、こだわりの1着 |
| **判断基準**: 「1年後に覚えている？」→ Yes=拡張 / 「覚えてないけど毎日の質に関わる」→ 維持 |

#### 拡張 vs ノイズ

| 拡張 | ノイズ |
|---|---|
| 意識的に選んでいる | 無意識・惰性で使っている |
| 「これにお金を使いたい」と思える | 後から「別にいらなかったな」と思う |
| 週末のワイン、旅行、記念日の食事 | なんとなくウーバー、セールの衝動買い |
| **判断基準**: 「使う前に楽しみにしていた？」→ Yes=拡張 / 「気づいたら使っていた」→ ノイズ |

#### 迷ったときの問い

どの分類か迷ったら、以下を順番に自問する：

1. **これがなかったら生活が成り立たない？** → Yes: 生存
2. **1年後に覚えていそう？** → Yes: 拡張
3. **毎日の暮らしの質に関わる？** → Yes: 維持
4. **上のどれにも当てはまらない** → ノイズの可能性

### 拡張の優先順位の考え方

拡張は月4.5万円。無限には使えないので、「何に使うか」を意識的に選ぶ必要がある。

#### 拡張を選ぶ3つの軸

1. **2人の共有度** — 2人で共有する体験か、片方だけの楽しみか。どちらも大事だが、バランスを取る
2. **記憶の残り方** — 「あのとき楽しかったね」と後から話せるものほど拡張としての価値が高い
3. **頻度と単価のバランス** — 毎週の小さな拡張（週末のカフェ）と、年数回の大きな拡張（旅行）を両方持つ

#### 計画的な拡張 vs 突発的な拡張

| 計画的 | 突発的 |
|---|---|
| 旅行、記念日の食事、欲しかった家具 | 偶然見つけた素敵な店、友人からの急な誘い |
| 特別費や旅行積立から出すか検討 | 拡張の月予算内ならOK。超えるなら「来月から調整」で対応 |
| **事前に2人で話す** | **事後に「これ良かったね」と確認する** |

突発的な出費が拡張かノイズかは、**事後に振り返ったとき「良い体験だった」と思えるかどうか**で判断する。

### 大きな出費の判断フレームワーク

想定外の出費や、高額な買い物をするときの話し合い方。

#### Step 1: 分類する
- 「これは4つのうちどれ？」を2人で話す
- 壊れた家電の買い替え → 生存。悩まず買う
- 前から欲しかったもの → 拡張 or 維持。Step 2へ

#### Step 2: 出どころを決める
- 特別費の積立（年40万円）から出せるか？
- 拡張の月予算から分割で吸収できるか？
- どちらでもなければ、貯蓄を一時的に調整するかを相談

#### Step 3: 合意する
- 片方が「欲しい」、片方が「高い」と思ったとき → **分類の言葉で話す**
  - 「それ維持？拡張？」「拡張だとしたら今月の残り枠はいくら？」
  - 感情（「高い！」「でも欲しい！」）ではなく構造（「拡張の枠に入るかどうか」）で議論する
- 合意できないとき → **1週間寝かせる**。1週間後にまだ欲しければ拡張、忘れていたらノイズ寄り

### なぜこの分け方にするのか

1. **我慢前提にしない** — 節約が目的ではなく、楽しく続けられる設計
2. **夫婦で話しやすい** — 「それノイズかもね」と構造で話せて、感情的になりにくい
3. **幸せを守る** — 削るのはノイズだけ。楽しみは削らない

## 月次振り返りセッション

月1回、あゆさんと一緒に。クレカ・PayPay等の明細をスマホで見ながら、AIが対話形式でヒアリングする。

### 目的

1. **バランスの確認** — 4分類の配分が大体取れているかを確認する
2. **分類力を育てる** — 「これはどの分類？」と迷う体験を通じて、2人の判断基準を育てる
3. **拡張を意識する** — ちゃんと楽しめたか、来月は何を楽しみたいかを話す時間にする

### セッションの流れ

AIは以下の順番でヒアリングする。選択式UIを使い、気軽に答えられる形式にする。

#### Step 1: 今月の全体感（体感ベース）
- 「今月のお金の使い方、全体的にどうでした？」（選択式: いい感じ / 使いすぎた感 / 節約しすぎた / よく分からない）
- 「あゆさんはどう思いますか？」（同上）

#### Step 2: ジャンル別ざっくりヒアリング
明細を見ながら、ジャンルごとに大まかな金額と感触を聞く。

1. **食費**（スーパー・外食・ウーバー等）
   - 「食費は全部でどのくらいでした？」（選択式: 3万以下 / 3〜5万 / 5〜8万 / 8万以上 / 分からない）
   - 「その中で『これはノイズだったな』と思うものはありますか？」（自由記述）
   - 「逆に『これは良い拡張だった』と思う食事はありますか？」（自由記述）

2. **日用品・生活費**（ドラッグストア・Amazon等）
   - 「日用品系はどのくらいでした？」（選択式: 1万以下 / 1〜2万 / 2〜3万 / 3万以上）
   - 「衝動買いや"なんとなく"の買い物はありましたか？」

3. **外出・交通・交際費**
   - 「外出や人と会うのに使ったお金はどのくらい？」（選択式: 1万以下 / 1〜3万 / 3〜5万 / 5万以上）
   - 「楽しかった・思い出に残った使い方はありますか？」

4. **その他で大きかったもの**
   - 「今月、いつもと違う大きな出費はありましたか？」（自由記述）
   - あれば4分類のどれに当たるか一緒に判定

#### Step 3: 分類を深める対話
ヒアリングの中で「これどっちだろう？」と迷ったものを1〜2個ピックアップして掘り下げる。

AIが問いかける例：
- 「ウーバーで3回頼んだとのことですが、3回とも同じ気持ちでしたか？ 疲れて仕方なく頼んだのと、あえて美味しい店を選んだのでは分類が変わりますよね」
- 「このワイン代、週末に楽しんで飲んだ分と平日になんとなく飲んだ分、分けるとしたらどのくらいの比率ですか？」
- 「この買い物、1週間後に『買ってよかった』と思っていますか？」

**目的**: 正解を出すことではなく、2人で「うちにとってのノイズってこういうことだよね」という共通認識を育てること。

#### Step 4: 4分類の振り分け（AIが整理）
ヒアリング内容をもとに、AIが大まかに4分類に振り分けて提示する。

```
今月のざっくり配分（推定）:
生存:  約○万円（目安10万円）
維持:  約○万円（目安4万円）
拡張:  約○万円（目安4.5万円）
ノイズ: 約○万円（目安1.5万円以内）
合計:  約○万円（目安20万円）
```

#### Step 5: 拡張の振り返りと来月の計画
- 「今月、良い拡張はありましたか？思い出に残った使い方は？」
- 「来月、楽しみにしていること・お金を使いたいことはありますか？」
- 「逆に、先月の拡張で『思ったほどでもなかった』ものはありますか？」（→ 次回から維持 or ノイズに再分類）

**拡張を計画的に楽しむ**ことで、突発的なノイズに流されにくくなる。

#### Step 6: 気づきと合意
- 「今月の気づきは？」（2人それぞれ）
- 「来月に向けて2人で決めておきたいことはありますか？」（大きな出費の予定、拡張の優先順位など）
- AIからも所感を1つ（押し付けず、構造的な気づきのみ）

### 記録

振り返りの結果は `振り返りログ/YYYY-MM.md` に月別で保存する。

### 大事なルール

- **正確さより感触を大事にする** — 1円単位の家計簿ではない。「大体こんな感じ」でOK
- **ノイズの判定は本人たちが決める** — AIが「それはノイズです」と断定しない
- **感情的にならない構造** — 「ノイズかもね」と言い合える関係を支える仕組み
- **楽しみ（拡張）を確認するのも大事** — ちゃんと楽しめたかを振り返る時間でもある
