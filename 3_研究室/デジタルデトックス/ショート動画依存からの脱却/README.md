# ショート動画依存からの脱却

ショート動画依存からの脱却と、デジタルとの健全な距離感を設計する。

## 現状の課題

- 1日中スマホを触っている
- 起床後すぐ1〜2時間ショート動画を視聴（脳のゴールデンタイム浪費）
- 就寝前1時間もショート動画
- 「やめたいのにやめられない」状態
- ドーパミン依存のメカニズムが関与

## 構造的な理解

これは意志力の問題ではなく、ドーパミン報酬系の問題。
「気合いでやめる」は効かない。物理的な遮断・環境設計で対処する。

## 考えたいこと

- 起床後のスマホに触らない仕組み（物理的に遠ざける等）
- 就寝前のスマホ代替行動（読書、ストレッチ等）
- スクリーンタイムの計測と可視化
- 「Appleメモ依存からの距離の取り方」（雑記帳より）との接続
