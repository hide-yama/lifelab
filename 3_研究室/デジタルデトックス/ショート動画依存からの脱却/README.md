# ショート動画依存からの脱却

**ステータス**: 未着手 — 課題の整理まで完了、具体的な施策の実行はこれから

ショート動画依存からの脱却と、デジタルとの健全な距離感を設計する。

## 現状の課題

- 1日中スマホを触っている
- 起床後すぐ1〜2時間ショート動画を視聴（脳のゴールデンタイム浪費）
- 就寝前1時間もショート動画
- 「やめたいのにやめられない」状態
- ドーパミン依存のメカニズムが関与

## 構造的な理解

これは意志力の問題ではなく、ドーパミン報酬系の問題。
「気合いでやめる」は効かない。物理的な遮断・環境設計で対処する。

### ショート動画が脳に与える影響（2026-03-12 調査）

**結論: ショート動画は「休憩」ではなく、脳にとっては重労働に近い。**

#### なぜ脳が消耗するのか

1. **高速タスクスイッチング** — 15〜60秒ごとに全く違うコンテンツに切り替わる。1時間で数百回のコンテキストスイッチが発生し、認知資源を大量に消費する（Monsell, 2003）
2. **ドーパミンの搾取** — 「次は面白いかも」という予測不確実性がスロットマシンと同じ原理でドーパミンを最大放出。繰り返すと受容体が鈍化し、日常の穏やかな活動（読書、散歩）では満足できなくなる（Volkow et al., 2011）
3. **注意残余** — 視聴後に仕事に戻っても、直前の動画の断片が注意を奪い続ける（Leroy, 2009）

#### 1時間以上視聴した場合の影響

| 影響 | 自覚症状 |
|---|---|
| 認知資源の大幅消耗 | 頭がぼんやり、考えがまとまらない |
| ドーパミン基準値の低下 | 何をしても楽しくない、虚無感 |
| 持続的注意力の低下 | 長い文章が読めない、5分で飽きる |
| 意思決定力の低下 | 「もう何でもいい」状態 |
| 身体的倦怠感 | だるい、動きたくない |

#### 問診データとの一致（Day 1〜4）

睡眠の質・量が悪い日ほどショート動画の視聴量が増える傾向が確認された：
- Day 2（睡眠まあまあ）→ 動画中断あるが眠気なし
- Day 1・3（飲酒で睡眠の質低下）→ 眠気あり＋動画多め
- Day 4（睡眠5時間50分）→ 動画がっつり1時間以上（4日間で最多）

**睡眠不足 → エネルギー低下 → ショート動画で逃避 → さらに脳が消耗 → 悪循環**

### 見てしまった後の回復方法（2026-03-12 調査）

「ドーパミンデトックス」や「ドーパミンリセット」という概念は科学的には不正確。ドーパミンは脳の神経伝達物質であり、短期間の断食で「リセット」されるものではない（Cleveland Clinic; University of Reading）。ただし、高刺激からの離脱と認知資源の回復には科学的根拠のある方法がある。

#### 短時間でできること（5〜30分）

1. **自然の中を歩く（20分）** — 注意回復理論（ART: Kaplan, 1995）に基づく。自然環境は「柔らかな注意（soft fascination）」を誘発し、意図的注意の疲労を回復させる。公園を20分歩くだけで注意力と気分が有意に改善（Berman et al., 2008）。都市部の散歩よりも自然環境のほうが効果が大きい
2. **冷水で顔を洗う / 冷水シャワー（1〜3分）** — 交感神経を刺激し、覚醒水準を引き上げる。ダイブ反射（哺乳類潜水反射）により迷走神経が活性化し、副交感神経系のリセットにも寄与する
3. **身体を動かす（10〜15分）** — 軽い運動（ウォーキング、ストレッチ、スクワット）でBDNF（脳由来神経栄養因子）が分泌され、前頭前皮質の機能が回復する。ショート動画依存者において身体活動が自己制御力を媒介して依存を軽減するという研究結果あり（Frontiers in Psychology, 2025）
4. **マインドフルネス呼吸（5分）** — ボックス呼吸法（4秒吸う→4秒止める→4秒吐く→4秒止める）。前頭前皮質の活動を回復させ、衝動性を低下させる

#### 中期的に取り組むこと（数日〜数週間）

1. **睡眠の質の改善** — EEG研究により、ショート動画視聴後もベータ波・ガンマ波活動が持続し、注意の切り替え困難と認知負荷の増大が生じることが確認されている。就寝前のスクリーン使用を避け、メラトニン分泌を妨げないことが回復の基盤
2. **段階的に刺激の強度を下げる** — いきなり完全断絶ではなく、ショート動画→長い動画→ポッドキャスト→読書→散歩と段階的に移行する。脳の報酬系が穏やかな刺激で満足できるよう再調整される
3. **「退屈」に耐える練習** — 待ち時間にスマホを見ない、5分間何もしない練習。ドーパミン受容体の感受性は、過剰刺激をやめれば数日〜数週間で回復に向かう（神経可塑性による）

#### 仕事に戻るためのブリッジ手順

ショート動画を見てしまった後、いきなり難しい仕事に戻るのは逆効果。以下の手順で「橋渡し」する：

1. スマホを物理的に離す（別の部屋に置く）
2. 5分間何もしない or 深呼吸する
3. 紙に「今日やること」を1つだけ書く
4. その中で「最初の5分だけやる」と決めて着手する
5. ポモドーロ・テクニック（25分集中＋5分休憩）で再起動する

### 見始めないための予防策（2026-03-12 調査）

#### 物理的な環境設計（最も効果が高い）

1. **スマホを別の部屋に置く** — 物理的距離が最も強力な介入。Ward et al.（2017）の研究で、スマホが視界にあるだけで認知能力が低下することが確認されている。フリーランス/リモートワーカーは仕事中スマホを「玄関」「別の部屋」に置く
2. **グレースケールモードにする** — 画面を白黒にすることでSNSの視覚的報酬（カラフルなサムネイル）を無効化。研究では1日あたり平均38〜50分のスクリーンタイム削減が確認（Holte & Ferraro, 2020; Dekker & Baumgartner, 2024）。iOSではアクセシビリティ→ショートカットで簡単に切り替え可能
3. **アプリの削除 or ホーム画面からの除外** — TikTok等をホーム画面から消し、Appライブラリのみにする。もしくは完全削除してブラウザ経由のみにする（摩擦を増やす）
4. **スクリーンタイムの利用制限** — iOSのスクリーンタイム機能で1日の利用上限を設定。パスコードは信頼できる人に設定してもらうとより効果的
5. **通知の全オフ** — ショート動画アプリの通知を完全にオフ。通知は「きっかけ刺激（cue）」として依存ループのトリガーになる

#### 心理的テクニック

1. **If-Thenプランニング（実行意図）** — 「もしスマホに手が伸びたら、そのとき代わりに水を飲む」のように事前にルールを決める。メタ分析で行動変容に対する中〜大の効果サイズが確認されている（Gollwitzer & Sheeran, 2006）
2. **10分ルール** — 「見たくなったら10分待つ」。衝動は通常5〜10分でピークを過ぎて減衰する（サーフィング・ジ・アージ）。待っている間に衝動が消えることが多い
3. **コスト計算の可視化** — 「1時間のショート動画視聴 = 仕事の生産性2〜3時間分の損失」を紙に書いて作業机に貼る。注意残余と認知資源の回復に要する時間を含めた実質コスト
4. **トリガーの特定** — 自分がどの状況でスマホに手を伸ばすか記録する（退屈、疲労、ストレス、食後、起床直後）。トリガーごとに代替行動を設定する

#### 代替行動（休憩時間にやること）

「休憩のつもりで見始めて止まらなくなる」パターンへの対策は、休憩を「受動的な消費」から「軽い能動的行動」に置き換えること：

| 代替行動 | 時間 | 効果 |
|---|---|---|
| 散歩（できれば自然の中） | 10〜20分 | 注意回復・気分改善（ART） |
| ストレッチ・軽い筋トレ | 5〜10分 | 血流改善・覚醒度UP |
| 紙の本を読む | 10〜15分 | 持続的注意力の維持 |
| 音楽を聴く（受動的だが有限） | 5〜10分 | 気分転換・終わりがある |
| 手書きで何か書く | 5分 | 思考の整理・前頭前皮質の活性化 |
| コーヒー/お茶を淹れる | 5分 | 身体的ルーティンによるリセット |
| 窓の外を眺める | 3〜5分 | 微小な注意回復（自然の視覚刺激） |

**重要**: 休憩に「終わり」がある行動を選ぶ。ショート動画は「無限スクロール」設計のため終わりがない。「コーヒーを淹れる」「1章だけ読む」のように自然な終了点がある行動が望ましい。

#### フリーランス/リモートワーカー向けの特別な対策

1. **朝のルーティンにスマホを入れない** — 起床から最初の1時間はスマホに触らない。目覚まし時計をスマホ以外にする（アナログ時計 or スマートスピーカー）
2. **仕事専用デバイスの分離** — 可能ならPC（仕事用）とスマホ（私用）を明確に分離。仕事中はスマホを機内モード or 別室にする
3. **作業環境の設計** — 「この机に座ったら仕事モード」という空間的な条件付けを作る。スマホは作業エリア外に配置
4. **ポモドーロ・テクニックの導入** — 25分作業＋5分休憩（休憩は上記の代替行動から選ぶ）。休憩にスマホを使わないルールにする
