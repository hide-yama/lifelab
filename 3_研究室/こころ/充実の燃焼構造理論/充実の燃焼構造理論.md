# 充実の燃焼構造理論 
## 完成版ドラフト

## 命題

本理論が記述対象とするのは、善い生一般ではなく、**主観的一致としての充実感**である。

充実感とは、主体が仮説として採用した **Want to be（どんな人間として在りたいか）** に照らして、**Want to do（やりたいこと）**、**Should do（現実の側から到来する要請のうち、いま応答すべきだと主体が引き受けたもの）**、そしてそれを成立可能にする **Can do（やれること）** が整合した行為を遂行し、その体験が自らの方向性にかなっていたと**事後的に認識される**ときに生じる現象である。

設計できるのは充実感そのものではなく、**方向性の仮説と三条件の整合**である。充実感は事後的にしかわからない。しかし、整合した燃焼が起こりやすい条件は設計できる。その結果として、充実感は事後的に認識される。

本理論が扱うのは、外的条件そのものではなく、**それらが主体にどう現れ、どう引き受けられているか**である。したがって本理論における充実感は、客観的な成功や道徳的正しさではなく、主観的一致として記述される。

---

## 見取り図

```text
上位レンズ： Want to be（火守り）
              ↓ 照らす
三条件：    Want to do（燃料）× Should do（酸素）× Can do（炉）
              ↓ 燃焼
結果：      火（充実感は事後的に認識される）
              ↓ 時間
生成物：    物語（連鎖反応）
外乱：      風（偶然）
```

Want to be は上位レンズであり、他の三条件と同列ではない。  
Want to be が方向を与え、Want to do が駆動を与え、Should do が現実への応答を与え、Can do が成立可能性を与える。  
この四者が時間の中で整合したとき、燃焼が生じ、その燃焼が自らの方向性にかなっていたときに、充実感が事後的に認識される。

---

# 第一章　構造

## 1. Want to be —— 方向を決めるもの

Want to be とは、**どんな人間として在りたいか**を示す長期的な方向性であり、更新可能な仮説である。  
それは肩書きや結果目標ではない。何を美しいと感じ、どの代償なら引き受けられ、どんな燃え方を自分のものとして選ぶか。その選好の束が Want to be を形づくる。

Want to be は固定的本質ではない。人はまず方向を仮置きし、その方向に照らして行為を選び、実際の燃焼を経験し、その経験によって方向の精度を更新する。  
したがって Want to be は、**先に完全に発見されるものではなく、燃焼を通じて鍛えられる仮説**である。

Want to be は状態ではなく方向である。  
「成功したい」「有名になりたい」は結果であって、それ自体では Be ではない。  
Be が問うのは、何を得たいかではなく、**どういう仕方で生きたいか**である。

### Be と Should の境界
Be は人を述べ、Should は場面を述べる。  
Be は「どういう人でいたいか」を定め、Should は「この状況で何に応答すべきか」を定める。  
Be は時間をまたいで残り、Should は要請が去れば形を変える。

### Be の歪み
Want to be はしばしば歪む。典型は三つある。

- **外部内面化型**：他人の価値観を自分の方向性だと思い込む  
- **防衛型**：傷つかないための回避を理想としてしまう  
- **快楽最適化型**：短期的快楽に引きずられ、方向が欲求に従属する

### Be を見分ける基準
よい Be 仮説には、次の特徴がある。

- 代償を含めて選び取れている
- 他者の承認がなくても残る
- 一時的気分ではなく、反復して立ち戻る

---

## 2. Want to do —— 駆動を生むもの

Want to do とは、主体の内側から湧き上がる駆動である。  
欲望、好奇心、創造衝動、試してみたさ、没頭したさ。  
それは行為を始めさせるエネルギーであり、火における燃料にあたる。

ただし、欲求は「自分のものらしく見える」だけでは足りない。  
**欲求の本物性は、出自そのものではなく、その欲求がどれだけ自分の土壌に同化され、代償込みで引き受けられているか**で決まる。

### 欲求の出自
欲求の出自は大きく三つに分けられる。

- **自生**：自分の経験や気質から自然に育ったもの  
- **移植**：他者や文化から受け取ったが、自分の土壌で育て直されたもの  
- **外来**：偶然持ち込まれたもの、まだ同化されていないもの

重要なのは、どこから来たかより、**それが自分の燃料として生きているか**である。

### 欲求の状態変化
欲求は時間とともに変わる。  
まだ熟していない欲求もあれば、かつて強かったが朽ちた欲求もある。  
一瞬だけ強く燃える欲求もあれば、長く静かに燃える欲求もある。  
したがって診断すべきは「やりたいことがあるか」だけでなく、**その欲求はいまどんな状態にあるか**である。

---

## 3. Should do —— 応答すべきこと

Should do とは、**現実の側から到来する要請のうち、主体がいま応答すべきだと認識し、引き受けたもの**である。  
ここでいう「現実」とは、社会、他者、状況、対象、身体などを含む。

Should は道徳説教ではない。  
それは「べき」という圧力の記号ではなく、**行為に具体的な重みを与える応答課題**である。

人が子どもを迎えに行くのは、子どもという他者がいるからである。  
パン生地を寝かせるのは、対象がそう要請しているからである。  
休息をとるのは、身体がそう要請しているからである。  
ゲームの局面に集中するのは、その局面が応答を求めているからである。  
Should は社会的義務に限られない。**現実に応答するための引き受け**全般を指す。

この定義により、Should は充実感の成立に**必要**である。  
ただし、その厚みには幅がある。  
職業的責任のように厚い Should もあれば、素材や身体や一瞬の局面に応じるような薄い Should もある。  
**Should は必須だが、常に重いとは限らない。**

### Should の由来
Should の由来は次の四つに整理できる。

- **制度・社会**：法、役割、市場、慣習
- **特定他者**：依頼、期待、必要
- **状況**：締切、事故、機会、文脈
- **対象・身体**：素材の論理、技法の要請、身体の限界や回復要求

### Should の引受け度
同じ要請でも、引き受け方は異なる。

- **強制**：拒否権がないと感じている
- **同調**：理由を持たず従っている
- **選択**：コストを理解したうえで引き受けている
- **同一化**：自分の実践の一部として統合されている

ここでいう Should の「純度」とは、道徳的高さではなく、**どれだけ反省的に引き受けられているか**を意味する。

### Be と Should の関係
外部から来た要請が、そのまま Should になるわけではない。  
要請は Be に照らして選別され、引き受けられたときに Should になる。

> 現実の要請 → Be による選別 → 引き受けられた Should

したがって Should は外圧そのものではない。  
また、自己規範そのものでもない。  
自己規範は Be 側の方向性であり、Should はそれが場面に降りてきた応答である。

---

## 4. Can do —— 成立可能性の条件

Can do とは、燃焼を成立させる条件の総体である。  
それは単なる才能やスキルではない。体力、時間、集中力、回復力、経験、制御力、環境適合性まで含む。

Can do を機能で分けると、少なくとも五つある。

- **強度**：どれだけの負荷に耐えられるか  
- **容量**：どれだけ同時に扱えるか  
- **排熱**：蓄積した熱を逃がせるか  
- **効率**：同じ投入でどれだけ燃焼できるか  
- **制御**：火力を上げ下げし、止まれるか

「分かっているのに動けない」「続けたいのに続かない」は、意志の弱さよりもまず Can の問題として読むべきである。  
Do と Should があっても、Can が受け止められなければ火は立たない。  
逆に、Can が整うだけで急に燃え始めることもある。

Can の問題には二種ある。

- **物理破綻**：単純に強度・容量が足りない  
- **設計破綻**：配分や制御や排熱が悪い

理論上、これは重要な区別である。  
前者には縮小や回復が必要であり、後者には設計変更が効く。

---

## 5. 代償と引受け

Be、Do、Should、Can のすべてには代償がある。  
何かを選ぶことは、別の何かを手放すことである。

- Be は、ある生き方を選ぶことで別の生き方を手放す  
- Do は、ある欲求を生きることで他の選択肢を狭める  
- Should は、ある要請に応答することで時間や自由を差し出す  
- Can は、ある能力を使うことで、その能力に縛られる

したがって、充実感は単なる一致ではない。  
**代償を納得して引き受けた一致**である。  
この引受けが欠けていると、形式上は燃えていても、充実感は歪む。

---

# 第二章　動態

## 1. 物語 —— 燃焼が燃焼を呼ぶ

一回の整合は、一回の充実感を生む。  
しかしそれが反復されると、経験が理解を深め、理解が次の選択を変え、次の選択が新たな燃焼を生む。  
この連鎖が生まれたとき、充実感は単発の出来事ではなく、**物語**になる。

物語は構造の前提ではない。  
構造が時間の中で反復された結果として立ち上がる生成物である。  
だが一度物語が生まれると、それは Be・Do・Should・Can を逆に変え始める。

---

## 2. 燃焼状態

燃焼状態は大きく四つに分けられる。

- **消火**：三条件がほぼ立っていない  
- **くすぶり**：Do はあるが、Should か Can が足りない  
- **瞬間燃焼**：一時的に整合するが、反復や物語に至らない  
- **安定燃焼**：整合が持続し、物語が回り始めている

診断とは、どの状態にいるかを見極めることである。  
処方とは、その状態に応じて、どこを調整するかを選ぶことである。

---

## 3. 充実感の複数軸

充実感に序列はない。  
ただし、その性質を記述する軸はある。

- **持続時間**：瞬間的か、持続的か  
- **関係性**：一人で完結するか、他者を含むか  
- **物語統合度**：単発か、連鎖に入っているか  
- **Be 一致度**：方向性との一致が高いか  
- **Should の厚み**：応答すべき要請がどれほど厚く関与しているか

ここで重要なのは、Should の厚みが薄い充実感も成立するが、**Should そのものがゼロで成立するわけではない**ということだ。  
一人の遊びにも、身体、対象、局面、時間の要請がある。  
酸素は薄いことがあるが、無いことはない。

---

## 4. 病理

燃焼は成功だけでなく、失敗や歪みも持つ。主要な病理は次の通りである。

- **過熱**：Do と Should は本物だが、Can を超えている  
- **借り物燃焼**：Do が十分に同化されていない  
- **酸欠**：応答すべき現実への感受性が痩せている  
- **点火失敗**：Do と Should はあるが Can が受け止められない  
- **有毒燃焼**：整合はあるが、自己や他者に有害な形で燃えている

病理の厄介さは、**歪んだ燃焼も燃える**ことにある。  
だからこそ診断が必要になる。

---

# 第三章　営み

## 1. 火守り

火守りとは、メタレベルの自己である。  
燃焼の最中に巻き込まれながらも、その燃焼を観察し、配置し、調整し、必要なら縮小し、必要なら消し、新しい火を試す主体である。

火守りの仕事は、正解を発見することではない。  
**仮説としての Be を更新し続けること**である。

---

## 2. 複数の火

人は一つの火だけを生きていない。  
仕事、家庭、友情、趣味、身体、学び。  
それぞれに別の Do・Should・Can があり、別の燃焼がある。

この理論は基本的に一つの火を単位として記述する。  
しかし人生全体の充実感は、**複数の火の総体**として火守りが認識する。  
したがって問題は、単一の理想の火を探すことではなく、**火同士の配分と干渉**を見極めることでもある。

---

## 3. 診断と処方

本理論の実践的価値は、次の三段階にある。

### 第一段　診断する
- いまはどの燃焼状態にいるか  
- 問題は Be、Do、Should、Can のどこにあるか  
- Can のどの機能が詰まっているか

### 第二段　求める火を定める
- どんな燃え方を望んでいるか  
- どの代償なら引き受けられるか  
- その火は一時的か、持続的か、関係的か

### 第三段　処方する
- Do を見直す  
- Should の引受け方を見直す  
- Can を育てる、修繕する、配分し直す  
- 過去の灰をかき出す  
- 必要なら待つ  
- Be 仮説を更新する

本理論は数値測定の理論ではない。  
それは、言語化しにくい感覚に輪郭を与えるための**診断理論**である。

# 第四章　修正条件と射程

## 1. この理論が説明するもの

本理論は、主観的一致としての充実感を記述する。  
推奨可能性や道徳的善さそのものは判定しない。

| | 推奨しやすい | 推奨しにくい |
|---|---|---|
| **充実感がある** | 天職、成熟した実践、納得した責任 | 加害的だが主観的には充実している生 |
| **充実感がない** | 報われないが尊いケア、やむをえない献身 | 惰性、麻痺、搾取 |

本理論が判定するのは、左列ではなく**上段と下段**である。  
右列の是非は、別の理論の仕事である。

---

## 2. 倫理について

本理論は**倫理理論ではない**。  
有害な実践であっても、主観的に整合していれば充実感は成立しうる。  
それを否定しないことは、倫理的に容認することとは別である。

この峻別を拒むと、理論はきれいになる代わりに現実から目をそらす。  
本理論は、その誘惑を退ける。

---

## 3. 修正条件

本理論は自然科学的な厳密反証を主張しない。  
その代わり、以下のような事例が安定して確認されるなら、理論修正が必要である。

### 修正条件1  
主体が長期にわたり高い充実感を報告しながら、  
- 方向性を持たず  
- やりたいこともなく  
- 応答すべき要請も認識せず  
- 成立可能性も認めていない  
にもかかわらず、理論的読み替えなしにその充実感が持続する場合。

### 修正条件2  
充実感が、整合によってではなく、**持続的不整合そのもの**から生じると示される場合。  
たとえば、矛盾、裂け目、自己分裂こそが充実感の中核であり、整合はむしろ副次的だと示される場合。

### 修正条件3  
欲求や方向性を意図的に手放す修行的実践が、理論的翻訳なしに深い充実感を安定して成立させる場合。  
この場合、本理論は充実感を説明する基礎語彙そのものを見直す必要がある。

---

## 4. 射程の限界

本理論は「燃えたい」「うまく燃えられない」「燃えていたものを取り戻したい」と感じる人のための理論である。  
したがって、そもそも燃焼を価値としない生き方は、本理論の射程外にある。

静かに在ることそれ自体を価値とする生、燃えないことを選ぶ時期、水のように流れ、満ち、潤す在り方。  
それらをこの理論で無理に説明しようとすると、理論は暴力になる。

理論に限界があることは、理論の欠陥ではない。  
限界を知らないことが欠陥である。

---

## 結語

充実感とは、仮説としての Want to be に照らして、Want to do・Should do・Can do が時間の中で整合したとき、その燃焼が自らの方向性にかなっていたと事後的に認識される現象である。

人は完成された一致の中に住むのではない。  
方向を仮置きし、欲求を育て、現実の要請に応答し、成立可能性を整え、その結果を見てまた方向を更新する。  
この往復運動こそが、火守りの営みである。

充実感は所有物ではない。  
それは、偶然に左右されながらも、なお設計しうる条件の中で、繰り返し立ち上がる火である。
