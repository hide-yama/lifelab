# noteでの収益化リサーチ用プロンプト

## プロンプト1: 成功事例のリサーチ

```
noteで「自己理解」「生き方」「人生の指針」系のコンテンツを有料販売し、しっかり収益化できている個人クリエイターを調べてください。

特に知りたいこと:
- 具体的なクリエイター名とそのnoteアカウント
- どのような切り口・コンセプトでコンテンツを出しているか
- 有料記事の価格帯（単品 / マガジン / メンバーシップ）
- 無料記事と有料記事の使い分け方
- フォロワー数と有料コンテンツの販売実績（スキ数などから推定）
- SNS（X/Instagram/YouTube）との連携方法

対象は「ビジネス系インフルエンサー」ではなく、個人の体験や思想から独自のフレームワークや哲学を構築して発信している人に絞ってください。

また、note以外でも（Kindle出版、Brain、Tipsなど）同系統のコンテンツで収益化に成功している事例があれば併せて教えてください。
```

## プロンプト2: コンテンツ戦略の設計

```
以下のコンテンツをnoteで収益化したいと考えています。マーケティング戦略を提案してください。

【コンテンツの概要】
- テーマ: 人生の時間を「火」に例えて分類する独自フレームワーク
- 5つの場（焔・炭・灯・灰・炉）と20の火勢で、「なぜ頑張っているのに報われないのか」「好きだったことが義務になるのはなぜか」「休んでいるのに疲れるのはなぜか」といった日常のモヤモヤを構造的に説明する
- 正解を教えるのではなく「今あなたがどこにいるか」を自覚させるコンパス型のコンテンツ
- 書籍としての全体構成は6章。第1〜5章で常識をひっくり返し、第6章でフレームワークの全体像を種明かしする

【現在の状態】
- 第1章（4節）とまえがきは執筆済み
- フレームワーク自体の理論設計は完了
- 発信実績はまだゼロ

【知りたいこと】
1. 無料記事で集客→有料で回収する場合、何を無料にして何を有料にすべきか
2. 単品記事 vs 有料マガジン vs メンバーシップ、このコンテンツに最も合う課金モデルはどれか
3. 最初の1記事目は何を出すべきか（最もバズりやすい切り口）
4. noteのSEOやおすすめアルゴリズムで有利に働くタイトル・タグ・投稿頻度の目安
5. 0→1000フォロワーまでの具体的なロードマップ
```

## プロンプト3: 競合ポジショニング分析

```
「人生の時間の使い方」「自己理解」「生きづらさの構造的な解明」をテーマにしたコンテンツは、日本の出版・note・SNS市場でどのようなポジションが取られていますか？

以下の観点で分析してください:

1. 既存の競合コンテンツ（書籍・note・YouTube）の主な切り口を分類してください
   - 例: 時間管理術系、自己啓発系、心理学系、哲学系、エッセイ系など

2. それぞれの切り口の飽和度（レッドオーシャンかブルーオーシャンか）

3. 以下の独自ポジションは市場にどの程度空いていますか？
   - 「正解を教えない」スタンス（コンパス型）
   - 独自の比喩体系（火・炭・灰など）で日常のモヤモヤを構造化
   - 自分の生活実験データを公開しながら理論を検証していくプロセス公開型

4. 類似のポジションで成功している人がいれば、その人との差別化ポイントは何になりうるか

5. このコンテンツが最も刺さりそうなターゲット層（年齢・職業・心理状態）の仮説
```
