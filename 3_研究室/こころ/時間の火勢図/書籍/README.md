# 書籍執筆ガイドライン

## 基本情報

- **仮タイトル**: 「あなたの人生がうまくいかない本当の理由」
- **理論本体**: `../統合理論.md`（人生における燃焼設計理論）
- **構成案**: `../書籍構成案.md`

## 構造

- 第一〜五部（第1〜21章）: 症状の描写。読者の体験として書く
- 第六部（第22〜27章）: 理論の開示。前半の全てがここでつながる

## 文体

### やること

- 短い文と長い文のリズムを作る
- 冒頭は具体的な情景から入る。「その人は〜」のように
- 火のメタファーは日常の比喩として自然に使う（床板を剥がす、煙ばかりで燃えない、等）
- 読者に「これは自分のことだ」と思わせる具体性
- 一話は短く。エピソードとして読める分量

### やらないこと

- 「この章で伝えたいのは」「ここで重要なのは」等のメタ的な言い回し
- 「答えはないが」「正直に言えば」等の野暮な前置き。まえがきで宣言済み
- 理論用語を使わない（燃焼条件、炉の素材、欠損パターン等）。第六部まで出さない
- 同じことを言い換えて繰り返さない。一回で伝える
- 「〜しましょう」「〜してみてください」等の指導口調

### 火のメタファーについて

前半では理論を匂わせない。読者は「うまい比喩だな」くらいに思って読み進める。第六部で初めて、それが比喩ではなく構造だったと気づく。この体験を最大化するために、前半では火の比喩を「理論として」使わない。

## 各章と理論の対応

| 部 | 統合理論の対応箇所 |
|---|---|
| 第一部（絶好調の顔で壊れる） | 火の病理学「一見うまくいっているが危うい火」 |
| 第二部（ダメじゃない） | 「一見うまくいっていないが実は健全な状態」＋炉の適合 |
| 第三部（静かな窒息） | 「いつの間にかうまくいかなくなった火」 |
| 第四部（抜け出せない） | 「違うと思っているのにやめられない火」 |
| 第五部（何も始まらない） | 七つの欠損パターン |
| 第六部（構造の全体像） | 三要素・燃焼条件の三層・物語・求める火の姿 |

## 執筆の進め方

- 書籍構成案の各章タイトルに沿って1章ずつ書く
- ファイル命名: `第一部_第1章.md`, `第一部_第2章.md`, ...
- 書いた後は必ず読み返し、重複・蛇足・メタ的表現を削る
