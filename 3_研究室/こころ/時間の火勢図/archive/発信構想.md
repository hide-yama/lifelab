# 発信構想（火の理論ベース）

## 理論の核

人が充実するメカニズムを、燃焼の三角形で説明する。

- **燃料** = やりたいこと（内発的エネルギー。種類によって燃え方が違う）
- **酸素** = やるべきこと・需要（量だけでなく質がある。何が自分を燃やしているか）
- **やれること** = 火を成立させる一切の条件と能力。以下の三層で構成される
  - 第一層：炉の形状（気質・特性。火との適合度）
  - 第二層：炉の素材（健康・人間関係・経済基盤。構造材）
  - 第三層：火守りの技術（着火・維持・調節・修繕を含む動的な管理能力）
- **自分** = 火の一つ上のメタレイヤー。火を営む存在
- **火は刹那**。人生は無数の火の連なり
- **求める火は人によって違う**。蝋燭、囲炉裏、溶鉱炉、松明
- **酸素には質がある**。純酸素、通常の空気、フッ素的酸化剤——何で燃やされているかで副産物が変わる

目的は測定ではなく**可視化**。「自分の火は今どうなっているか」を直感的に掴むためのフレームワーク。

---

## 二刀流戦略: note × 本

### note = 火の診察室

- 1記事1症状。「あ、これ自分だ」と思わせる
- 燃焼の三角形を全部説明する必要はない。火の比喩で「今起きていること」を描写する
- 読者が複数読むうちに「全部つながっている」と気づく
- 理論の全体像はnoteには出さない

### 本 = 火守りの教科書

- noteで「症状」に触れた人が「なぜこうなるのか」「どうすればいいのか」を知りたくなったとき
- 燃焼の三角形、やれることの三層構造、火守りの技術、求める火の種類——理論の全体像を体験的に伝える
- 「ああ、あのnoteの話はこういう構造だったのか」が本でしかできない体験

### 流れ

```
note（症状の描写）→ 共感 → 本（メカニズムと処方）→ 自分の火を見つめ直す
```

## コンセプト

**「あなたの火は、ちゃんと燃えていますか？」**

ターゲット: 頑張っているのに満たされない人。忙しいのに充実しない人。好きだったことが楽しくなくなった人。やめたいのにやめられない人。——全部、火の問題として説明できる。

ポジショニング: 「こうすれば幸せになれる」を教えるのではなく、**自分の火の状態を見る目**を渡す。求める火は人によって違う。だから処方も違う。万人共通の正解はない。でも「今何が起きているか」を理解する言葉はある。

## 書籍構成案（仮タイトル「火守りの手帖」）

### 第1章　火がつかない

3要素のどれかが欠けている状態を、読者自身の体験として描く。

**1-1「やりたいことがあるのに、動けない」**
燃料と酸素はある。でも発火点に届かない。やれることが閾値に達していない。能力が足りないのか、炉の形と火の相性が悪いのか、環境が熱を奪っているのか、それとも触媒が見つかっていないだけか。

**1-2「できることはある。求められてもいる。でも何も燃やしたいものがない」**
酸素とやれることはある。燃料がない。有能で求められているのに空っぽ。最も見えにくい苦しさ。

**1-3「誰にも求められていない情熱」**
燃料とやれることはある。酸素がない。くすぶっている。作品を作っても読まれない。技術があっても発揮先がない。

**1-4「3つが揃っても、火がつかないことがある」**
炉の形状と火の適合の問題。同じ3要素でも、自分の気質や特性と噛み合っていなければ燃えない。「能力が低い」のではなく「組み合わせが悪い」だけかもしれない。能力には「上げる」以外に「合わせる」という解がある。

---

### 第2章　燃えているのに、おかしい

一見うまくいっているが、内部に問題がある火。

**2-1「自分の床板を剥がして燃やしている」**
健康、家族、信頼、貯蓄——炉の素材を引き剥がして燃料にしている。火は明るい。でも足元が崩れていく。構造材燃焼の病理。

**2-2「誰かの薪で燃えている火」**
親の期待、上司のビジョン、社会の正解。供給が止まった瞬間に何を燃やせばいいかわからなくなる。自分の内発的エネルギーではなかった。

**2-3「求められすぎて、炎が暴れている」**
酸素過多。風が強すぎる火。求められるままに応え続けて燃え尽きる。

**2-4「よく燃えるけど、煙が有毒」**
フッ素的な酸素。搾取的な需要、依存的な承認、支配的な関係。激しく燃えるが毒が出る。やめられないのは、暗闘で他に灯りがないから。解決策は「消す」ことではなく、別の火を先に灯すこと。

---

### 第3章　いつの間にか消えかけている

気づかないうちに進行する火の衰え。

**3-1「同じ場所で燃やし続けた結果」**
密閉空間の酸素枯渇。同じ環境、同じ需要に応え続けて、酸素が痩せていく。火が弱まってから気づく。

**3-2「過去の灰が、新しい火を塞いでいる」**
成功体験、ルーティン、実績。燃焼の副産物が燃料を覆い、酸素との接触を遮断する。

**3-3「薪が腐っていることに気づかない」**
燃料の変質。やりたかったはずのことが、いつの間にか中身が変わっている。見た目は同じだが、もう燃えない。

**3-4「消えたように見えて、まだ熱い」**
熾火。灰の下にまだ赤い炭がある。無理に薪を投げ込むと灰が舞って本当に消える。この状態の正しい対処は「そっと風を送る」こと。

---

### 第4章　火守りの技術

火を燃やし続けるための動的な技術。「やれること」の第三層。

**4-1「薪のくべ方」**
燃料の投入ペース。一度に大量にくべれば酸素が遮断される。少なすぎれば燃え尽きる。

**4-2「通気口の調節」**
酸素の流入を制御する。どの需要を受け入れ、どこから遮断するか。

**4-3「灰をかき出す」**
過去を手放す技術。灰をためると火が窒息する。

**4-4「炉を修繕する」**
休息、健康管理、関係の修復。炉の素材の修繕。火を燃やしていない時間に見えて、実は最も重要な火守りの仕事。

**4-5「小さな火から始める」**
火守りの技術は経験で磨かれる。いきなり大火を扱おうとせず、小さな火を何度も焚く。

---

### 第5章　あなたはどんな火を求めているか

求める火の種類によって、処方がまったく変わる。

**5-1「蝋燭の火」**
小さく静かだが、精密で美しい。一冊の本を照らすだけの光。この人に焚き火の技術を教えても意味がない。

**5-2「囲炉裏の火」**
家族や近しい人を温め続ける持続的な火。大事なのは燃料の質と灰のかき出し。

**5-3「溶鉱炉の火」**
極限の高温で何かを鍛え上げる。膨大な燃料と酸素を消費するが、それでしか作れないものがある。

**5-4「松明の火」**
自分が進む道を照らし、周囲にも光を見せる。環境が変わっても燃え続けられることが生命線。

**5-5「自分の火を定義する」**
上の4つは例示にすぎない。あなたの火は、あなたにしかわからない。

---

### 第6章　火を営む

理論の全体像を開示し、読者自身が使えるようにする。

**6-1「燃焼の三角形」**
燃料、酸素、やれること。3つ揃うと火がつく。どれが欠けても消える。

**6-2「やれることの三層」**
炉の形状——自分の気質と特性を知ること。炉の素材——構造材を守り、修繕すること。火守りの技術——経験で磨かれる動的な能力。

**6-3「火は刹那である」**
一つの火に固執しない。人生は無数の火の連なり。

**6-4「複数の火を営む」**
仕事の火、家庭の火、創作の火。燃料と酸素の配分。全体を見渡すメタレベルの自分。

## note記事の方針

- 第1〜5章の各節がそのまま1本のnote記事になる
- 理論の全体像（第6章）はnoteには出さない。本だけの体験
- 各記事は「症状の描写 → 火の比喩での解説 → 一つの問い」で構成
- 読者に「診断名」を渡すのではなく「自分で火を見る目」を渡す
