# 時間の火勢図

**ステータス**: 執筆中 — 書籍 第1章完了、第2章執筆中。noteでの発信準備も並行

グラッサー博士の「上質世界（Quality World）」を出発点に、人生の時間をどう分類・可視化し、意思決定と生活改善に使えるフレームワークを作る研究。

上質世界は「好き/好きじゃない」の二項対立だが、現実の時間にはもっと多くのグラデーションがある。それを構造化したい。

## 着地点

1. **フレームワークの完成** — 自分の時間を分類する実用的な地図
2. **共通言語としての活用** — 人生相談の基準、夫婦間の対話ツール、作戦立案の道具
3. **noteでの発信** — 「上質世界の先にたどり着いた独自フレームワーク」として記事化

## 研究の経緯

1. 選択理論心理学の「上質世界」の動画を見て概念を知る
2. 「上質世界だけでは現実の時間を分類しきれない」と感じる
3. 2軸4象限（内発/外発 × 滋養/消耗）のマトリクスを発案
4. ゾーン間の動的な力（火勢）を3つ定義
5. AIレビューで5つの課題が見つかる
6. ブラッシュアップ版の設計（v2）— 5つの場 + 暴走チェック + 関係性拡張
7. ネーミングを火のメタファーに統一（焔/炭/灯/灰/炉 + 火勢）
8. 火勢を20に拡張（5つの場 × 4方向 = 全移動パスの網羅） ← **今ここ**
