# 早起き版の1日のスケジュール

作成日: 2026-03-11

## 前提

- 起床 7:00 / 就寝 23:00（睡眠8時間）
- 仕事は9:00開始（9:15朝ミーティング）、定時17:50
- 朝のマスト: 身支度・朝食（45分）＋ 問診（30分）
- 夜は固定しない。日によって研究・読書・生活タスクを柔軟に

## スケジュール

```
7:00  起床
7:00  身支度・朝食（45分）
7:45  朝の問診（30分）
8:15  朝の余白（研究 or リラックス）
9:00  仕事開始（9:15 朝ミーティング）
      ─── 仕事 ───
17:50 定時・仕事終了
18:00 夕方の問診（15分）
18:15 夕飯の準備・夕飯
19:30 自由時間（研究・読書・生活タスクなど日による）
21:30 風呂
22:30 リラックス・就寝準備
23:00 就寝
```

## 現状との差分

- 起床: 9:00〜10:00 → 7:00（2〜3時間前倒し）
- 仕事終了: 18:30〜19:30 → 17:50（定時で切る）
- 朝に問診＋余白の時間が生まれる
